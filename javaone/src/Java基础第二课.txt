Java是个纯面向的对象语言
它的表现形式表现为:
1.把一组具有相同属性和方法的对象抽取分类，形成
Java所定义的类,类的关键字就是class,
注意c是小写。

--相同的属性，java是个强类型语言，决定在定义属性，就是变量
的时候，定义属性名称(参照标识符命令规范),定义属性的类型:
1.String class Java里面叫做引用类型，对象类型，复合类型

2.Java的基本数据类型:
  2.1整数类型  byte short int   long
  2.2浮点类型  float  double  默认类型是:double ,定义float 取值要加f
  2.3布尔类型  boolean
